# Posts and customers management API

> REST API developed using nodejs, express, and MongoDB.

## Setup Quick start

```bash
# Clone repository
git clone https://github.com/dhananjay5544/mongo-crud-api.git

# Install dependencies
cd mongo-crud-api
npm install

# Run server (:5000)
npm run dev

```
